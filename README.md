## Что это?

**Mnemosyne** — расширение для Chrome, которое переводит и озвучивает
YouTube-видео: берёт субтитры, переводит их через Claude AI и озвучивает перевод
нейронным голосом синхронно с видео — чтобы смотреть контент на любом языке, не зная его.

Никаких пауз на чтение. Никакой борьбы с кривым автопереводом субтитров. Просто смотри
и слушай на своём языке.

> ⚠️ **Alpha 0.0.1.** Базовый сценарий (перевод + нейронная озвучка + субтитры) работает.
> Возможны шероховатости качества перевода и артефакты в озвучке (`[СМЕХ]`, «э-э» и т.п.) —
> см. [планы после альфы](#-после-альфы).

## Возможности

- 🌍 **Любой язык**, доступный на YouTube — английский, японский, испанский, корейский и др.
- 🧠 **Перевод через Claude AI** — качество выше штатного автоперевода YouTube
- 🔁 **Два режима перевода** — готовый автоперевод YouTube или перевод через Claude API
- ⚡ **Перевод заранее, до воспроизведения** — без задержек и заиканий посреди фразы
- 🔊 **Нейронная озвучка (Edge TTS)** — голоса Microsoft Neural, работают «из коробки»
  без VPN и настройки (через общий прокси-Worker проекта)
- 🎚️ **Синхронизация с видео** — темп речи подстраивается на лету, чтобы не отставать;
  настраиваемый диапазон скорости, сдвиг, приглушение оригинала
- 📝 **Караоке-субтитры** — переведённые строки прямо на видео (предыдущая / текущая / следующая)
- 💾 **Умное кеширование** — повторный просмотр загружается мгновенно, перевод не тянется заново
- 🛠️ **Панель разработчика** — Live State Inspector, меню кэша, API Monitor и калькулятор
  стоимости для полной прозрачности

## Как это работает

```
Субтитры YouTube  →  Claude AI (перевод)  →  Edge TTS (нейронная озвучка)
                                                    ↓
                                      Караоке-субтитры + приглушённый оригинал
```

Mnemosyne читает уже существующую дорожку субтитров YouTube — без обработки звука и распознавания
речи. Текст уходит в Claude на перевод, возвращается и озвучивается синхронно с видео.

**Источник текста** — перехват реального запроса плеера к `/api/timedtext`: прямой `fetch`
по ссылке субтитров YouTube отдаёт пустой ответ без `pot`-токена, поэтому расширение берёт
уже валидный запрос самого плеера.

**Озвучка** — прямой эндпоинт Microsoft Edge из браузера недоступен (нужны «свои»
Origin/User-Agent + блокировки в части регионов). Синтез идёт через общий **Cloudflare
Worker** проекта: он вне зоны блокировки, повторяет протокол `edge-tts` и возвращает MP3.
Конечному пользователю настраивать ничего не нужно.

## Дорожная карта TTS-движков

| Движок | Статус | Качество | Заметки |
|---|---|---|---|
| **Edge TTS (Microsoft)** | ✅ Реализован (alpha) | ⭐⭐⭐⭐ | Нейронные голоса через Cloudflare Worker, работает из коробки |
| Web Speech (системный) | ✅ Доступен | ⭐⭐ | Локальные голоса браузера, без сети; запасной вариант |
| CosyVoice 3 | 🔜 В планах | ⭐⭐⭐⭐⭐ | Локальный, клонирование голоса |
| Qwen3-TTS | 🔜 В планах | ⭐⭐⭐⭐⭐ | Локальный, модель 0.6B, поддержка русского |
| Yandex SpeechKit | 🔜 В планах | ⭐⭐⭐⭐⭐ | Лучшие русские голоса на рынке |

TTS-движок абстрагирован за общим интерфейсом (`lib/tts.ts`) — замена не требует
переписывания логики планировщика.

## Технический стек

- **[WXT](https://wxt.dev)** — современный фреймворк для расширений Chrome с HMR
- **TypeScript** — строгий режим, без `any`
- **Claude AI** — перевод через [aiprimetech.io](https://aiprimetech.io)
  (drop-in замена официального Anthropic API, модель `claude-sonnet-4-6`)
- **Edge TTS** — нейронный синтез через Cloudflare Worker; Web Speech API — запасной движок
- **Manifest V3** — актуальный стандарт расширений Chrome

## Установка (alpha-версия)

Готовая сборка прикреплена к [релизу](https://github.com/GrayRK/Mnemosyne/releases) (файл `.zip`).

1. Скачай `mnemosyne-0.0.1-chrome.zip` из последнего релиза и распакуй в любую папку.
2. Открой `chrome://extensions` в Chrome (или браузере на его базе).
3. Включи **Режим разработчика** (тумблер справа вверху).
4. Нажми **Загрузить распакованное расширение** и выбери распакованную папку.
5. Открой popup расширения, вставь свой **API-ключ** (`aiprimetech.io`), выбери язык и голос.
6. Открой любое YouTube-видео с субтитрами и нажми кнопку перевода на плеере.

> Без API-ключа доступен режим **автоперевода YouTube** (переключается в popup) — перевод
> через Claude API требует ключа.

## Запуск из исходников (для разработки)

> Нужны Node.js 18+ и браузер на базе Chrome.

```bash
git clone https://github.com/GrayRK/Mnemosyne
cd Mnemosyne
npm install
npm run dev
```

WXT автоматически откроет Chrome с загруженным расширением.

Полезные команды:

```bash
npm run dev        # запуск Chrome с расширением (HMR)
npm run compile    # проверка типов (tsc --noEmit)
npm run build      # production-сборка (.output/chrome-mv3)
npm run zip        # упаковка для публикации (.zip)
```
